# Autonomous Prompt Optimizer

This repository contains a FastAPI backend and a Vite + React frontend for iteratively optimizing prompts.

Quick start (backend):

```bash
cd backend
python -m venv .venv
source .venv/Scripts/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Refer to DOCKER.md for Docker Compose-based local development.
